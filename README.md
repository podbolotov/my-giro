# Самостоятельная работа "Разработка одностраничного сайта"

Дисциплина: ОПЦ.14 Internet-технологии

# Общая информация

В рамках задания выполнена разработка сайта интернет-магазина гироскутеров My-Giro согласно предоставленным методическим рекомендациям.

В ходе работы над проектом я старался соблюсти все заложенные в методические рекомендации требования. 

Помимо основных (обязательных) требований реализованы следующие дополнительные указания:
- Для фильтра реализована возоможность выбора товаров всех типов.
- Вёрстка, по мере возможностей, адаптируется к разрешению 1024х1366px.
- Раздел "Наши преимущества" реализован в виде "карусели" с возможностью переключения отображаемого элемента посредством графических кнопок.

При разработке сайта использованы только HTML и CSS.\
JavaScript в проекте не применяется.

Отладка и тестирование произведены в браузере Google Chrome версии 142.0.7444.162 для macOS. 

Для репозитория проекта активирован модуль "GitHub Pages", вследствие чего просмотр сайта возможен из сети Интернет посредством [соответствующей ссылки](https://podbolotov.github.io/my-giro/) (podbolotov.github.io/my-giro). 

P.S. Выравнивание группы кнопок фильтра в каталоге намеренно сделано "по центру", а не "по левому краю родительского блока" по причине большей (по моему мнению) визуальной привлекательности такого решения.
